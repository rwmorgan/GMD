Unity Engine Guide

Industry-standard engine for 2D and 3D games. Uses C# scripting and has the largest game development community — and tutorial library — in the world.

2D + 3D C# scripting Free for students

When to Choose Unity

  • You want to make 2D or 3D games — Unity does both well
  • You're interested in a long-term career in games (Unity is used by thousands of studios)
  • You want access to a huge Asset Store of free and paid resources
  • You're comfortable learning C# scripting — or want to be
  • Your game needs physics, lighting, or 3D environments

Hardware Requirements

Minimum RAM8 GB
Recommended RAM16 GB
Storage (install)~10 GB
GPUDX10 compatible or better
OSWindows 10/11 or macOS 12+

Key File Types

  • .unity — Scene files
  • .cs — C# scripts
  • .prefab — Reusable game object templates
  • .meta — Asset metadata (never delete these)
  • .asset — Serialised Unity assets (materials, settings)

Sprite & Audio Import Specs

Sprites: Import as PNG. Set Texture Type to Sprite (2D and UI). For pixel art, set Filter Mode to Point (no filter) and Compression to None. Pixels Per Unit should match your game's resolution grid.

Audio: Import .wav or .ogg files. Set Compression Format to Vorbis for music (enable streaming), PCM for short SFX.

Getting Started

  1. Download Unity Hub from unity.com (free). Install the latest LTS version.
  2. Create a free Unity Personal licence — for students, personal use is always free.
  3. Create a new 2D or 3D project. Name it with your game title — no spaces.
  4. Set up your Assets folder with subfolders: Sprites, Audio, Scripts, Scenes, Prefabs.
  5. Work through the Unity Essentials pathway on Unity Learn before touching your own project.
