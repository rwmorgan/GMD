Adobe Creative Suite

Guides for using Adobe tools in a game development context. All tools below are available in your school lab via your Adobe license.

🖼 Photoshop — Sprites & Tilesets

Used in Tasks

Task 3.1 (sprites), Task 3.2 (tilesets). Photoshop is the right tool for raster/pixel art — bitmap images at a fixed resolution.

Key Settings for Game Art

  • New document: set pixels as unit, no colour profile
  • Pixel art: turn off anti-aliasing on the pencil tool
  • Use a limited colour palette (16–32 colours for retro style)
  • Save source as .psd with layers. Export as .png (transparent background).
  • Export: File → Export → Export As → PNG. Check "Transparency".

Sprite Checklist

  • Canvas size matches engine's required sprite size (e.g. 32×32, 64×64)
  • Background layer is transparent (not white)
  • Each sprite element on its own named layer
  • Exported as flat .png — NOT .psd
  • File named correctly (no spaces: player_idle.png)

Learn more: Adobe Photoshop Tutorials ↗

✏️ Illustrator — Vector Assets

Use Illustrator for assets that need to scale to any size (logos, UI icons, vector characters). Vectors stay crisp at any resolution. For pixel art, use Photoshop instead.

  • Set up an Artboard for each asset
  • Export: File → Export → Export for Screens → PNG at 1x, 2x if needed
  • For use in Unity/GMS2: export as PNG (not SVG — most engines don't import SVG directly)

🎵 Audition — Game Audio

Used in Tasks

Task 3.3 (audio design). Audition is used to record, edit, normalise, trim, and export game audio files.

Essential Audition Skills for Task 3.3

  1. Import: File → Open. Or drag file into the Files panel.
  2. Trim silence: Select the silence at start/end, press Delete.
  3. Normalise: Effects → Amplitude and Compression → Normalize. Set to -3 dB for SFX, -6 dB for music.
  4. Export: File → Export → File. Choose format: .wav (44.1kHz, 16-bit) for SFX. .ogg for music loops.
  5. Loop point: For background music, set loop region in the waveform editor.

Free Audio Sources (for Task 3.3)

  • freesound.org ↗ — Community SFX library. Check each file's licence (CC0 = free to use without attribution; CC BY = use with attribution).
  • OpenGameArt.org ↗ — Music and SFX specifically for games. Always check licence.
  • incompetech.com ↗ — Kevin MacLeod's royalty-free music. Requires attribution (text provided on site).
  • BFXR (browser) ↗ — Generate 8-bit style SFX instantly. CC0 — no attribution needed.

⚠ Always record what licence each file uses before downloading. You need this for your Asset Catalogue (Task 3.4).

🎬 Premiere Pro — Trailers & Video

Used for game trailers, screen recording edits, and presentation videos. Not required for any task but useful for the Task 6.8 game page if you want to include a trailer.

  • Screen record with Windows Game Bar (Win+G) or OBS (free)
  • Import recording into Premiere, cut to highlights, add music from your audio pack
  • Export: File → Export → Media. Format: H.264. Preset: Match Source - High Bitrate.

⚡ Adobe Express — Quick Design

Used for: Task 1.3 (infographic), Task 2.2 (concept pitch), Task 5.2 (presentation), Task 6.8 (game page). Adobe Express is free with your school Adobe licence and runs in the browser.

  • For infographics: choose an infographic template, replace text and icons with your content
  • For presentations: use the "Presentation" template category
  • Export: Download as PDF (for print/submission) or PNG (for embedding)
  • Access at express.adobe.com ↗
