Microsoft Office

Word, Excel, PowerPoint, and OneNote — used across every unit. Guides for the features you'll actually need.

📝 Word — Documents & GDDs

Required Word Skills (by task)

  • Heading Styles: Home tab → Styles panel. Apply Heading 1, 2, 3 to section titles. This enables the automatic Table of Contents.
  • Table of Contents: References → Table of Contents → Automatic Table 1. Update it (right-click → Update Field) whenever you add sections.
  • Page Numbers: Insert → Header & Footer → Page Number → Bottom of Page.
  • Tables: Insert → Table. Right-click header row → Table Properties → check "Repeat as header row at top of each page".
  • Export as PDF: File → Export → Create PDF/XPS. Use this for final submissions of infographics and concept docs.

APA References (quick guide)

Website: Author, A. A. (Year, Month Day). Title of article. Site Name. URL

Example: Smith, J. (2023, March 5). The history of the platformer genre. Game Developer. https://gamedeveloper.com/...

Use the References tab → Insert Citation → Manage Sources for automatic bibliography generation.

📊 Excel — Gantt Charts & Trackers

Building Your Gantt Chart

  1. Column A: Task names. Columns B onward: one column per week (label B1="Week 15", C1="Week 16" etc.)
  2. For each task, shade the cells in the week columns where that task is active. Use conditional formatting or manual fill colour.
  3. Colour code by category: green = design, blue = coding, orange = art, yellow = audio, red = testing.
  4. Add an "Owner" column and a "Status" column (Not Started / In Progress / Complete).
  5. Freeze the top row (View → Freeze Top Row) so headers stay visible when scrolling.

📑 PowerPoint — Presentations

Tips for Game Pitches & Final Presentations

  • One idea per slide. Don't put 8 bullet points on a slide. Say the thing, show the image.
  • Use a dark theme. Games look better on dark backgrounds. Design → Themes → choose a dark template.
  • Screenshots over descriptions. A screenshot of your game is worth 10 sentences about what it looks like.
  • Embed your font. File → Options → Save → check "Embed fonts in the file". Prevents font substitution on different machines.
  • Practice the demo separately. Have your game open in a window behind PowerPoint. Alt+Tab to switch during the presentation.

📒 OneNote — Portfolio & Journal

Setting Up Your Portfolio in OneNote

  1. Create a new Notebook: "Game Design Portfolio — [Your Name]"
  2. Create one Section per unit (Unit 1 through Unit 5)
  3. Inside each Section, create Pages for each task's evidence
  4. Drag screenshots, paste excerpts, embed links — OneNote handles mixed media well
  5. For your Major Project development journal: create a new Section called "Dev Journal" with one page per week. Date stamp every entry.

OneNote syncs automatically to OneDrive — your journal is always backed up.
