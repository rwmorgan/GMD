Unreal Engine Guide

AAA-quality 3D engine from Epic Games. Blueprints visual scripting means you can build complex behaviour without writing a single line of code.

No-code Blueprints High-end 3D Engine is free

When to Choose Unreal

  • You're making a 3D game and want the best possible visuals
  • You prefer visual scripting (Blueprints) over writing code
  • You've used Unreal before and are confident in it
  • Your lab computer meets the higher hardware requirements
  • You want to learn the engine behind Fortnite and most modern AAA games

Hardware Requirements

Minimum RAM16 GB
Recommended RAM32 GB
Storage (install)~60+ GB
GPUDedicated GPU (GTX 1080 / RTX class)
OSWindows 10/11 or macOS 12+

Scope warning: Unreal is the most demanding option in this course. Keep your Major Project concept small — one polished level beats five unfinished ones.

Key File Types

  • .uproject — Project file (open this)
  • .uasset — All assets (meshes, textures, Blueprints)
  • .umap — Level/map files
  • .cpp / .h — C++ source (only if using code)

Texture & Audio Import Specs

Textures: drag PNG into the Content Browser. Set Compression Settings to UserInterface2D for UI sprites. For pixel art, set Filter to Nearest in Texture settings.

Audio: import .wav files. Unreal converts to its internal format. Use Sound Cues (or MetaSounds in UE5) for complex audio behaviour.

Getting Started

  1. Install the Epic Games Launcher, then install Unreal Engine 5 from its Library tab — the engine itself is free.
  2. The install is 60+ GB: in class, use the lab machines where it is pre-installed.
  3. Create a new project from a template (Third Person, Top Down, Side Scrolling, First Person) with Blueprint selected, not C++.
  4. Learn the viewport controls first: right-mouse + WASD to fly, F to focus.
  5. Then start the official Your First Hour in Unreal Engine 5 course (linked below).
