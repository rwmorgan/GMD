ICT205114Individual · Phase 1 · Weeks 1–22

Computer Applications Level 2

5 criteria · 5 TCE credit points · Awards: EA / CA / SA / PA

TASC Course Page ↗ Assessment Matrix →

How ratings work: ICT205114 uses both C (Satisfactory) and A (High Standard) ratings. EA = 5×A. CA = 2×A + 3×C. SA = 5×C. PA = 2×C. A 'z' in any criterion results in a PA regardless of other ratings.

C1

Identify and apply basic terminology and concepts relating to computer hardware and software

▾
C — Satisfactory
  • Correctly identifies hardware components and describes their usage
  • Correctly identifies software types and formats (OS vs applications vs files; .docx, .pdf, .jpg, .png, .yyp, .unity, etc.)
  • Applies file management skills (open, save, close, delete, rename, duplicate, transfer)
  • Describes ways of dealing with data storage issues (cloud storage, backup, security)
A — High Standard
  • Correctly identifies hardware and software components for specific purposes and justifies the choice
  • Applies knowledge and understanding of hardware and software to effectively solve real-world problems
Assessed in: Task 1.1 Task 1.2 Task 4.1
C2

Identify social and ethical issues related to the use of computers

▾
C — Satisfactory
  • Describes social and ethical issues related to computers and digital technologies
  • Correctly describes concepts of intellectual property and copyright
  • Describes and complies with principles for using others' information, images, ideas or words
  • Correctly describes and complies with given OHS procedures
A — High Standard
  • Correctly identifies social and ethical issues in specific situations and justifies the choice
  • Accurately describes OHS issues in specific situations, their possible health/safety impact, and methods to avoid, remove or minimise adverse impact
Assessed in: Task 1.3 Task 3.3 Task 3.4 Task 5.1
C3

Collect and communicate ideas and information using digital technologies

▾
C — Satisfactory
  • Collects relevant information from a range of sources, including the internet
  • Selects and re-synthesises information into a logical presentation
  • Uses computer applications/digital media to clearly present ideas and information
A — High Standard
  • Assesses the relevance, accuracy and completeness of collected information
  • Considers and effectively addresses the needs of various audiences and/or stakeholders
Assessed in: 2.1 2.2 2.3 3.1 3.3 3.4 5.1 5.2
C4

Complete tasks and solve problems using computers and other digital technologies

▾
C — Satisfactory
  • Selects and uses software and hardware appropriate to a given task
  • Uses basic features of selected software packages
  • Uses 'help' features, manuals and/or on-line support to solve problems
  • Uses effective internet search strategies; navigates and uses web-based applications
A — High Standard
  • Uses advanced features of selected software packages
  • Uses advanced internet search strategies including Boolean searches
Assessed in: 3.1 3.2 3.3 4.1 4.2 4.3
C5

Plan and organise resources and activities to complete computer application tasks

▾
C — Satisfactory
  • Maintains task focus for agreed periods of time
  • Sets goals which are generally measurable, achievable and realistic, and follows given plans/directions
  • Reflects on progress towards meeting goals and articulates ways goals might be met in the future
  • Uses strategies as directed to perform tasks within established timelines
A — High Standard
  • Maintains task focus
  • Sets goals which are measurable, achievable and realistic, and plans effective actions
  • Reflects on progress, evaluates progress and plans future actions
  • Considers, selects and uses strategies to manage and complete activities within established timelines
Assessed in: 2.3 4.2 4.3 5.1 5.2
