ESC205114Individual · Phase 1 · Weeks 1–22

Essential Skills – Using Computers & the Internet Level 2

4 criteria · 5 TCE credit points · Awards: SA / PA. Satisfactory Achievement is accepted as the TCE 'everyday adult' standard for using computers and the internet.

TASC Course Page ↗ Assessment Matrix →

ESC ratings: C = Satisfactory Standard only (no A rating). SA requires C in all 4 criteria. PA requires C in 3 criteria. All assessment is internal (teacher-assessed). No external exam.

C1

Identify digital technology tools suitable for routine tasks

▾
C — Satisfactory
  • Correctly identifies routine tasks in everyday adult settings, including the workplace
  • Correctly identifies digital technology tools appropriate to such routine tasks
Context in this course
  • Routine tasks include: selecting between Unity/GMS2/Unreal for a given game type, choosing Photoshop vs Illustrator for sprite work, identifying hardware specs needed for game development
Assessed in: Task 1.1 Task 3.1
C2

Interpret information to access and use digital technology

▾
C — Satisfactory
  • Correctly interprets routine information and terminology from a range of sources
  • Uses such information to access and employ digital technologies for routine tasks
Context in this course
  • Interpreting engine documentation, export specifications, Creative Commons licence terms, Adobe software Help menus, APA citation guides
Assessed in: 1.1 1.2 2.1 2.3 3.3 4.1
C3

Follow procedures to perform given tasks, and review performance

▾
C — Satisfactory
  • Correctly follows given instructions, specifications and/or directions
  • Uses digital technologies to successfully complete routine tasks
  • Undertakes review of their performance in such tasks
Context in this course
  • Following task specs exactly (file naming, structure, format), self-review sections in every task, reflection documents, feedback response sections
Assessed in: 1.2 2.1 2.2 2.3 3.1 3.2 3.3 4.1 4.2 4.3 5.1 5.2
C4

Use digital technologies effectively, safely and productively

▾
C — Satisfactory
  • Follows given guidelines/procedures relating to healthy and safe use of digital technologies
  • Identifies and describes everyday potential hazards (trip, spill, electrical, personal health)
  • Describes and complies with principles for use of the internet and email
  • Describes safe and unsafe practices in the use of social media and related technologies
  • Describes and complies with principles for use of others' information, images, ideas or words
Context in this course
  • OHS for screen-based work, lab hazards, netiquette, safe sharing of game dev work online, copyright and CC licences for game assets, academic integrity in research
Assessed in: 1.3 3.3 3.4 5.1
